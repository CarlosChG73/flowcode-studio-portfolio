# Flowcode Studio Portfolio

Portafolio profesional de Carlos Chávez, Ingeniero en Sistemas Computacionales, orientado al desarrollo de software, soluciones digitales y creación de interfaces web responsivas.

El proyecto presenta perfil profesional, formación académica, áreas técnicas, proyectos, enlaces profesionales y un formulario de contacto funcional mediante backend serverless.

## Vista general

Flowcode Studio Portfolio es una aplicación web desarrollada con React y Vite para presentar información profesional de forma clara, organizada y accesible.

El sitio incluye:

- Perfil profesional.
- Formación académica y desarrollo profesional.
- Áreas técnicas y herramientas.
- Proyectos y soluciones digitales.
- Página interna para detalle de proyectos.
- Enlaces profesionales.
- Formulario de contacto con envío de correo.

## Tecnologías principales

- React
- Vite
- Tailwind CSS
- React Router
- Vercel Functions
- Nodemailer
- React Icons
- Sharp
- Git

## Características

- Diseño responsivo para móvil, tablet y escritorio.
- Modo claro y oscuro.
- Navegación interna por secciones.
- Página individual para detalle de proyectos.
- Organización de contenido mediante archivos de datos.
- Evidencia visual del proyecto principal.
- Imágenes optimizadas en formato WebP.
- Formulario de contacto conectado a backend serverless.
- Variables de entorno para proteger credenciales del servicio de correo.
- Estructura organizada por componentes, páginas, datos, recursos visuales y utilidades.
